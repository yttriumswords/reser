# AI Student Health Diagnostic System

A simple HTML project that lets a user enter symptoms, temperature, severity, and duration to generate a **Health Report**.

This is an educational project and does **not replace professional medical advice**.

---

## 🗂️ Project Files

- **reser2.html** — The main HTML page for the diagnosis form.
- **style.css** — CSS stylesheet that styles the page.
- **README.md** — This file, explaining the project.

---

## 📥 Setup

1. Clone or download this repository.
2. Make sure both `reser2.html` and `style.css` are in the same folder.
3. Open `reser2.html` in your web browser.

---

## 🚀 Usage

1. Enter your symptoms in the form.
2. Fill in temperature, choose severity, and input duration.
3. Click the **Run Diagnosis** button to show results.
